# 🛒 Retail Sales & Inventory Performance Analysis

![Python](https://img.shields.io/badge/Python-3.10-blue) ![Pandas](https://img.shields.io/badge/Pandas-2.0-green) ![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange)

## Overview
Analysis of multi-branch FMCG retail sales and inventory data to identify performance gaps and procurement opportunities. Built on 15+ years of hands-on retail management experience across Hypercheck and Shoprite/Checkers branches in KwaZulu-Natal.

## Business Questions Answered
- Which product categories generate the most revenue?
- Which branches are underperforming and why?
- Where are stockout risks highest?
- What seasonal trends should inform procurement planning?

## Tools & Libraries
| Tool | Purpose |
|------|---------|
| Python / Pandas | Data wrangling & aggregation |
| Matplotlib / Seaborn | Data visualisation |
| NumPy | Numerical simulation |
| Jupyter Notebook | Interactive analysis |

## Key Findings
- Fresh Foods is the top revenue-generating category
- Beauty and Stationery have stockout rates exceeding the 20% threshold
- Revenue peaks in Q4 — promotional planning recommended
- Chatsworth branch underperforms — flagged for operational review

## How to Run
```bash
git clone https://github.com/aaronleebobby/retail-sales-analysis.git
cd retail-sales-analysis
pip install pandas numpy matplotlib seaborn jupyter
jupyter notebook retail_sales_analysis.ipynb
```
